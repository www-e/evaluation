# Project Overview

## Application Name
E-commerce Food Ordering Platform

## Purpose
This is a full-stack e-commerce application built with Next.js that allows users to browse food items, add them to a cart, and place orders. The platform includes an administrative dashboard for managing products, categories, and orders.

## Technology Stack
- **Frontend**: Next.js 16.1.0 (React framework)
- **Backend**: Next.js App Router with Server Actions
- **Database**: MySQL with Prisma ORM
- **Authentication**: Firebase Authentication (Phone-based)
- **File Upload**: UploadThing for image uploads
- **Styling**: Tailwind CSS with custom theme
- **State Management**: Zustand for client-side cart state
- **Validation**: Zod for server-side validation

## Key Features
1. **User Authentication**: Phone number-based authentication using Firebase
2. **Product Catalog**: Browse products organized by categories
3. **Shopping Cart**: Client-side cart with persistent storage
4. **Order Processing**: Create and manage orders
5. **Admin Dashboard**: Manage products, categories, and view analytics
6. **Responsive Design**: Mobile-first responsive layout
7. **Image Upload**: Product and category image management
8. **Search & Filter**: Product search and filtering capabilities

## Application Flow
1. User visits the homepage and browses products
2. User can register/login using phone number authentication
3. User adds products to cart and proceeds to checkout
4. Admin manages products, categories, and views order analytics

## Project Status
This is an MVP (Minimum Viable Product) version of the e-commerce platform with core functionality implemented.
